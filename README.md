# Introdução ao R para Estudantes de Biologia

Um material introdutório em português voltado a estudantes de graduação em Ciências Biológicas que desejam começar a utilizar **R** para análise de dados, visualização gráfica e organização de informações científicas.

Este repositório foi pensado especialmente para quem nunca programou antes e quer aprender de forma clara, prática e aplicada à Biologia.

---

## Sobre o projeto

R é uma das principais ferramentas utilizadas em diversas áreas da Biologia, como Ecologia, Evolução, Genética, Sistemática, Bioestatística, Conservação, etc.

Aprender R pode ajudar os biólogos a organizar dados de laboratório ou campo, automatizar tarefas repetitivas, criar gráficos de qualidade e reproduzir análises científicas.

Apesar disso, muitos estudantes têm pouco contato inicial com programação durante a graduação. Este material busca reduzir essa barreira por meio de explicações acessíveis e exemplos próximos da realidade biológica.

---

## Público-alvo

Este material é recomendado para estudantes de graduação em Biologia que são iniciantes em programação.

---

## Objetivos

Ao longo deste material, você será capaz de:

- Instalar R e RStudio
- Entender a interface básica do programa
- Executar comandos simples
- Criar objetos e vetores
- Importar planilhas de dados
- Explorar conjuntos de dados
- Produzir gráficos básicos
- Aplicar R em exemplos biológicos simples
